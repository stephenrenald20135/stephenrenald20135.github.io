Losing important emails can be stressful, especially when they’ve been permanently deleted from the Trash or Deleted Items folder in Outlook. Fortunately, there are several methods to recover these emails, depending on your Outlook setup and backup availability.

Methods to Recover Emails That Have Been Deleted From Trash in Outlook

We have 5 distinct methods here to recover emails that have been deleted from Trash in Outlook. Lets take a look at all of them individually.

1. Recover from "Recoverable Items" Folder

Outlook retains deleted emails in the Recoverable Items folder (formerly "Dumpster") for a limited time before permanent deletion.

Steps:

  1. Open Outlook and go to the Deleted Items folder.
  2. Click Recover Deleted Items from Server (for Exchange) or Recover Deleted Items (for Outlook 365/2019).
  3. Select the emails and click Restore.

Note: This method works only if the emails haven’t been purged from the Recoverable Items folder.

2. Restore from Outlook AutoArchive or Backup

If you have AutoArchive enabled or a PST backup, you may recover deleted emails from there.

Steps:

  1. Go to File > Open & Export > Import/Export.
  2. Select Import from another program or file and choose your backup PST.
  3. Follow the prompts to restore emails.

3. Use Windows File Recovery (For Deleted PST)

If your Outlook PST file is deleted, Windows File Recovery can help retrieve lost PST file.

Steps:

  1. Download Windows File Recovery from Microsoft Store.
  2. Run the tool with the appropriate command to scan for PST files.
  3. Recover and import the PST into Outlook.

4. Check Email Server or Exchange Backups

For business accounts, administrators may have server backups.

Steps:

  • Contact your IT admin to restore emails from an Exchange or Office 365 backup.
